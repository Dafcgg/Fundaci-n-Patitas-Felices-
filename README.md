# 🐾 Fundación Patitas Felices - Sistema Publicitario Web



## Descripción del Proyecto



Patitas Felices es una maqueta web estática desarrollada para promover la labor de rescate, cuidado, rehabilitación y adopción de animales en situación de vulnerabilidad.



El sitio busca ampliar la presencia digital de la fundación mediante una experiencia visual atractiva, accesible y responsive, permitiendo a los visitantes conocer la organización, visualizar animales disponibles para adopción y encontrar diferentes formas de colaborar.



El proyecto fue desarrollado utilizando únicamente **HTML5 y CSS3 nativo**, simulando elementos interactivos mediante animaciones y estilos CSS, sin emplear JavaScript.



---



## Objetivos



* Dar visibilidad a la misión y visión de la fundación.

* Mostrar animales disponibles para adopción.

* Facilitar la comunicación con posibles adoptantes, donantes y voluntarios.

* Presentar una interfaz moderna, accesible y adaptable a distintos dispositivos.

* Implementar componentes visuales interactivos utilizando únicamente CSS.



---

## Características Implementadas



### Página de Inicio



* Banner principal con logo, nombre y slogan.

* Botón Call To Action (CTA).

* Carrusel automático de imágenes utilizando CSS.

* Secciones destacadas de la fundación.

* Accesos rápidos a:



  * Adopción

  * Donaciones

  * Voluntariado

  * Contacto



### Misión y Visión



* Diseño responsive.

* Distribución lateral en pantallas grandes.

* Apilamiento automático en dispositivos móviles.



### Galería de Rescates



* Mosaico de imágenes responsive.

* Carrusel secundario animado con CSS.



### Catálogo de Adopción



* Más de 8 tarjetas de animales.

* Información incluida:



  * Fotografía

  * Nombre

  * Edad aproximada

  * Tamaño o raza

  * Estado (Disponible / Adoptado)

  * Botón "Ver Perfil"



### Perfil de Animal



* Imagen destacada.

* Historia y descripción.

* Características generales.

* Botón de solicitud de adopción.

* Sección de animales relacionados.



### Formulario de Contacto



* Campos obligatorios identificados visualmente.

* Simulación de validación mediante CSS.

* Tipos de consulta:



  * Adopción

  * Donación

  * Voluntariado

  * Información General

  * Otro



### Footer



* Nombre de la fundación.

* Información de contacto.

* Derechos de autor.

## Instrucciones de Ejecución



1. Clonar el repositorio:



```bash

git clone https://github.com/Dafcgg/-Fundaci-n-Patitas-Felices.git

```



2. Ingresar a la carpeta del proyecto:



```bash

cd patitas-felices

```



3. Abrir el archivo principal:



```bash

index.html

```



O utilizar una extensión como **Live Server** en Visual Studio Code para una mejor visualización.
